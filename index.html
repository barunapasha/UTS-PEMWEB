S-MART

S-MART (Student Marketing Art Technology) adalah organisasi mahasiswa yang berdedikasi untuk membantu para mahasiswa mengembangkan keahlian mereka dalam dunia pengembangan web, khususnya front-end development. Kami percaya bahwa setiap mahasiswa memiliki potensi untuk menjadi developer handal melalui pembelajaran praktis, kolaborasi, dan exposure terhadap teknologi terkini.

Nilai dan Tujuan

Creative Excellence

Mengedepankan kreativitas dan keunggulan dalam setiap karya, dengan fokus pada user experience dan estetika.

Collaborative Growth

Membangun komunitas yang saling mendukung melalui sharing knowledge dan kolaborasi antar anggota

Skill Enhancement

Meningkatkan kemampuan teknis mahasiswa dalam front-end development melalui program pelatihan terstruktur.

Industry Connection

Membangun jembatan antara mahasiswa dengan industri melalui networking dan exposure terhadap proyek nyata.

Apa yang Kami Tawarkan

FrontCraft Academy

Program pembelajaran intensif selama 3 bulan yang fokus pada fundamental front-end development. Peserta akan belajar HTML5, CSS3, JavaScript, dan framework modern seperti React.
Program ini mencakup:
Workshop mingguan dengan praktisi Proyek portfolio individual Code review session Mentoring one-on-one

S-MART Project Lab

Program kolaboratif dimana anggota berkesempatan mengerjakan proyek nyata untuk klien atau internal organisasi.
Program ini menawarkan:
Pengalaman kerja tim dalam proyek real Exposure ke project management tools Review dan feedback dari senior developer Kesempatan networking dengan industri
